# Api Crud
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/ArthurSilv4/apiCrud/blob/master/LICENSE.txt) 

## Sobre o projeto

Api crud é um projeto desenvolvido durante um final de semana para aprimorar meus conhecimentos em C#, .Net e entity framework.

O objetivo foi criar uma api que realiza operações de crud em um banco de dados sqlite.

## Tecnologias utilizadas
- C#
- Asp.net
- entity framework
- Sqlite

## Como executar o projeto

>Pré-requisitos: .Net Core 8.0


 Clonar repositório:
```
git clone https://github.com/ArthurSilv4/apiCrud.git
```

 Executar o projeto:
```
dotnet run
```

 Localhost:
``` 
https://localhost:7057/swagger/index.html
```
